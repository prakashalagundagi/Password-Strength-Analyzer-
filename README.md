🚀 Features

- 🔍 Password Strength Analysis
- ⏱️ Time-to-Crack Estimation
- 💡 Smart Suggestions for Strong Passwords

⚙️ Installation & Setup

1️⃣ Clone Repository

git clone https://github.com/your-username/password-analyzer-android.git

2️⃣ Open in Android Studio

- Click Open Project
- Select the folder

3️⃣ Add Dependency

implementation 'com.nulab-inc:zxcvbn:1.2.0'

4️⃣ Run the App

- Connect Android device / Emulator
- Click ▶ Run
